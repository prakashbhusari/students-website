Project Title: Students Website (Static Website Project)
1. Overview

This project is a simple static website designed for students to explore academic information, courses, gallery images, and contact details.
The website contains five web pages:

Home

About

Courses

Gallery

Contact

All pages are built using basic HTML5 and CSS3.
No backend or database is required.

2. Folder Structure
students-website/
│
├── index.html
├── about.html
├── courses.html
├── gallery.html
├── contact.html
├── style.css
└── images/          (optional folder for gallery images)

3. Technologies Used

HTML5

CSS3

Basic JavaScript (optional)

GitHub Pages / Netlify / Local Server for deployment

4. Features

Responsive navigation bar

Clean and simple UI

Easy-to-read content structure

Gallery section with placeholder images

Contact form (non-functional in static version)

5. How to Run the Website Locally

Download or clone the project folder.

Open the folder.

Double-click on index.html.

The website will open in a browser.

6. Deployment Instructions
Option 1: GitHub Pages

Create a repository on GitHub.

Upload all project files.

Go to Settings > Pages.

Select main branch and save.

GitHub will generate a live URL.

Option 2: Netlify

Login to Netlify.

Drag and drop your project folder into the dashboard.

Netlify will deploy it and provide a URL.

Option 3: Local Server (XAMPP)

Install XAMPP.

Copy your project folder to htdocs.

Start Apache.

Open http://localhost/students-website/.

7. Usage Notes

Ensure file names are correctly linked (case-sensitive).

index.html must be the homepage.

Use relative paths for images and CSS.

Keep style.css in the root folder to avoid path issues.

8. Future Enhancements (Suggested for Students)

Add JavaScript for form validation.

Add animations using CSS.
